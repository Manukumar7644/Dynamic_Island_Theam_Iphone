# 📱 Dynamic Island Theme Animation

A beautiful **Dynamic Island style animation** inspired by modern smartphone UI.  
This project demonstrates how the Dynamic Island expands to show incoming call details using **HTML and CSS animations**.

It simulates a real mobile call interaction where the island expands and displays the caller profile, name, and call action buttons.

---

## ✨ Features

- 📱 iPhone style UI design
- 🎨 Smooth CSS animations
- 👤 Caller profile preview
- 📞 Accept and Reject call buttons
- 💡 Clean and simple UI interaction
- ⚡ Pure HTML & CSS (No JavaScript)

---

## 🖥️ Preview

Dynamic Island expands when you hover over the device and displays call details like:

- Profile image
- Caller name
- Call accept button
- Call decline button
